# NomadGratis

NomadGratis is a mobile-first community platform for discovering and verifying free roadside and traveler resources.

The platform helps travelers, commuters, explorers, RV users, cyclists, and local communities locate useful public resources such as:

* Free air for tires
* Free EV charging
* Water refill stations
* Public parks
* Libraries
* Free WiFi
* Rest stops
* Trailhead parking
* Bike repair stations
* Free museums and attractions
* RV dump stations
* Public parking
* Free vacuums

Production:
https://nomadgratis.com

---

## Features

### Community-Verified Resources

Users can:

* add locations
* verify resources
* report outdated or incorrect data
* help maintain freshness and accuracy

### Mobile-First PWA

NomadGratis is designed primarily for mobile use with:

* installable PWA support
* offline-friendly behavior
* Android and iPhone support
* geolocation
* responsive map UX

### Structured Resource System

The platform uses:

* category-driven templates
* structured attributes
* configurable hours
* trust/freshness indicators
* moderation tooling
* scalable category architecture

### Discovery & Search

Users can:

* search by location/category
* browse nearby resources
* filter by availability and verification status
* access SEO-friendly category and detail pages
